# Bootstrap Workshop 6

Practice tasks for learning Bootstrap 5.3 components.

## Files

| File | Purpose |
| ---- | ------- |
| `index.html` | All five Bootstrap tasks plus About / Contact / Practice sections |
| `styles.css` | Custom styles layered on top of Bootstrap |
| `script.js` | Small JS demo (button click logging, smooth scroll, helpers) |
| `image.jpg` | Image used in the Task 4 card |

## Tasks covered

1. **Responsive Navbar** — `navbar-expand-lg` collapsing into a toggler below `lg`.
2. **Grid Layout** — `col-md-4 col-12` so 3 columns become 1 on mobile.
3. **Buttons** — primary / secondary / success / danger / warning / info.
4. **Card** — `card` with `card-img-top`, title, text, and CTA button.
5. **Alerts** — warning / success / danger / info using `alert-*` classes.

## Run it

Open `index.html` in any modern browser. No build step.
Bootstrap CSS and the bundled JS (with Popper) are pulled from the official CDN:

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```
