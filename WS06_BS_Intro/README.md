# Bootstrap Workshop 6 — Charlles Webpage Project

This project converts the original personal website into a responsive Bootstrap 5.3 website.

## Pages

- `index.html` — Home page with responsive navbar, hero section, Bootstrap alert, button group, card grid, and image card.
- `about.html` — Biography page with Bootstrap grid, image, cards, list group, and alert.
- `interests.html` — Interests page using Bootstrap cards, badges, alerts, and responsive columns.
- `contact.html` — Contact page with cards, buttons, alert, and Bootstrap form controls.
- `styles.css` — Custom styling layered on top of Bootstrap.
- `script.js` — Small JavaScript helpers for button logging and smooth scrolling.

## Workshop 6 requirements covered

1. Bootstrap 5.3 is included using the CDN.
2. The navbar uses `navbar`, `navbar-expand-lg`, `navbar-toggler`, and `collapse`.
3. The layout uses `container`, `row`, `col-*`, `g-*`, and responsive grid classes.
4. Buttons use Bootstrap variants such as `btn-primary`, `btn-secondary`, `btn-success`, and `btn-outline-primary`.
5. Cards use `card`, `card-body`, `card-img-top`, `card-title`, and `card-text`.
6. Alerts use Bootstrap alert classes such as `alert-warning`, `alert-info`, and `alert-success`.
7. The contact form uses `form-label`, `form-control`, and Bootstrap spacing utilities.

## How to run

Open `index.html` in a browser. Keep all files in the same folder so the images and CSS load correctly.
